# 💰 Financeiro Dashboard - CNPJ

Sistema de gestão financeira completo para acompanhamento e análise de finanças do CNPJ.

## 🎯 Objetivo
Ser um **gestor financeiro pessoal** que ajuda em:
- Análise de fluxo de caixa
- Tomada de decisões estratégicas
- Otimização de processos (BPO)
- Integração com consultor contábil

## 📁 Estrutura

```
financeiro-dashboard/
├── api/                    # Backend - Processamento de dados
│   ├── routes/            # Rotas da API
│   └── controllers/        # Lógica de negócio
├── frontend/              # Dashboard - Visualização
├── data/                  # Armazenamento de dados
│   ├── extratos/         # Arquivos de extrato (CSV/Excel)
│   └── processed/        # Dados processados
├── docs/                  # Documentação
└── SQUADS.md             # Integração com squads de estratégia
```

## 🚀 Funcionalidades (MVP)

- [ ] Upload de extratos bancários
- [ ] Processamento automático de dados
- [ ] Dashboard com métricas principais
- [ ] Gráficos de fluxo de caixa
- [ ] Categorização de transações
- [ ] Relatórios de análise

## 📊 Squads Integradas
- Gestor Financeiro
- BPO (Business Process Optimization)
- Consultor Contábil (on-demand)

## 🔧 Tech Stack
- Backend: Node.js + Express
- Frontend: HTML/CSS/JS (integrado ao admin existente)
- Data: JSON + CSV
