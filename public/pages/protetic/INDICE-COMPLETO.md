## 🎨 DESIGN DECISIONS EXPLICADAS

### Por que AOS.js?
✓ Lightweight (~7KB)  
✓ Performance (Intersection Observer)  
✓ WordPress-friendly  
✓ Sem conflito com Elementor  

### Por que CSS Variables?
✓ Fácil customização no Elementor  
✓ Consistent design system  
✓ Dark mode pronto (future)  

### Por que Grid 3col (mobile → 1col)?
✓ Melhor UX em mobile  
✓ Menos scroll horizontal  
✓ Leitura mais natural  

### Por que Tagline em Hero?
✓ Reforça novo posicionamento  
✓ Conexão emocional  
✓ Diferencia de concorrentes  

### Por que 7 steps no fluxo?
✓ Detalhe que faz diferença  
✓ Transparência com cliente  
✓ Reduz dúvidas
