    Brand Foundation

    O estúdio Rodrigo Roncolato é especializado em fotografia corporativa estratégica e brand strategy. A marca reflete profissionalismo, minimalismo e impacto visual através de uma paleta escura com destaque em amarelo vibrante. Cada elemento do design system foi cuidadosamente pensado para comunicar confiança, modernidade e expertise em personal branding.

    Valores Visuais

    • Escuro: Fundo #0A0A0A — sofisticado, moderno, reduz fadiga visual
    • Amarelo: Acento #F5C518 — energético, profissional, atrai atenção
    • Branco: Texto #F0F0EB — legibilidade, clareza, contraste
    • Cinzas: Hierarquia visual — 6 tons para diferentes contextos

    Regras de Uso — Logo

    • Espaço mínimo: 40px ao redor do logo (margin/padding)
    • Tamanho mínimo: 80px (desktop), 60px (mobile)
    • Nunca: Distorça, rotacione ou altere cores
    • Dark bg: Use logo branco (horizontal ou icon)
    • Light bg: Use logo preto (icon ou vertical)
    • Mobile: Prefira icon ou logo vertical

    8px Grid System

    REGRA DE OURO: Todos os tamanhos, espaçamentos, fontes e dimensões devem ser múltiplos de 8px. Isso garante consistência, harmonia visual e facilita escalabilidade em qualquer breakpoint.

    ✓ SEMPRE múltiplos de 8
    8px, 16px, 24px, 32px, 40px, 48px, 56px, 64px, 80px, 96px, 120px, 128px...

    Botões

    Cinco variantes de botão cobrindo todos os casos de uso. Cada variante tem estados default, hover e active documentados.

    Primary Button
    class="btn-primary"
    Amarelo fundo (#F5C518), preto texto. Usado para CTAs principais. Hover reduz opacity a 0.85.
    Outline Button
    class="btn-outline"
    Borda cinza, texto cinza. Hover: borda + texto branco. Secundário ou complementar.
    Dark Button
    class="btn-dark"
    Preto fundo, branco texto. Hover: borda branca. Contraste máximo.
    Filter Button
    class="btn-filter" / class="btn-filter active"
    Toggle com estado active. Default: borda cinza. Active: amarelo bg + preto texto.

    Motion & Animação

    Easing functions e durations padrão para manter consistência nas transições.

    Easing Curves

    --ease (Standard)

    cubic-bezier(.25, .46, .45, .94)

    Para transições suaves, scroll reveals, fade-ins

    --ease2 (Sharp Deceleration)

    cubic-bezier(.76, 0, .24, 1)

    Para efeitos snappy, hover states, cursor growth

    Durations Padrão

    0.3s: Hover states, border transitions
    0.4s: Custom cursor expand (8px → 56px)
    0.5s: Service card bottom border animation
    0.8s: Scroll reveal (.rv elements)
    1.2s: Skill bars animation (about page)
    24s infinite: Marquee strip scroll
    32s infinite: Testimonials carousel auto-scroll
