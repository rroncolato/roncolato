## 🎯 Exemplo: Criando uma Proposta do Zero

```bash
# 1. Criar pasta
mkdir -p "PROPOSTAS COMERCIAIS/projetos/silva-fotografia/v1"

# 2. Copiar template
cp "PROPOSTAS COMERCIAIS/templates/template-proposta.html" \
   "PROPOSTAS COMERCIAIS/projetos/silva-fotografia/v1/index.html"

# 3. Criar pasta de assets
mkdir "PROPOSTAS COMERCIAIS/projetos/silva-fotografia/v1/assets"

# 4. Abrir o HTML e customizar:
#    - [NOME DO CLIENTE] → Silva Fotografia
#    - [Tipo de Serviço] → Sessão de Fotos de Gestantes
#    - Preencher todas as seções
#    - Adicionar imagens em assets/

# 5. Passar pelo checklist (CRÍTICO!)
# → Sem erros ortográficos
# → Imagens OK
# → Links funcionam
# → Responsivo mobile
# → Cliente aprovou

# 6. Copiar para final/
cp -r "PROPOSTAS COMERCIAIS/projetos/silva-fotografia/v1" \
      "PROPOSTAS COMERCIAIS/projetos/silva-fotografia/final"

# 7. Deploy (DEPOIS DE CLIENTE APROVAR!)
cp -r "PROPOSTAS COMERCIAIS/projetos/silva-fotografia/final" \
      "public/proposta/silva-fotografia"
git add public/proposta/silva-fotografia
git commit -m "feat: proposta comercial - silva-fotografia"
git push

# 8. ✅ PRONTO! 
#    URL: rroncolato.com.br/proposta/silva-fotografia
```
