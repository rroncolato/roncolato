═══════════════════════════════════════════════════════════════════════════════
  💡 DICAS IMPORTANTES
═══════════════════════════════════════════════════════════════════════════════

1️⃣  MANTENHA O YELLOW (#F5C518)
   Este amarelo é a identidade visual do Roncolato!
   Mesmo na versão LIGHT do admin, ele deve ser o accent principal.

2️⃣  USE 8px GRID PARA TUDO
   Spacing: 4, 8, 16, 24, 32, 40, 48, 64px
   Isso garante consistência e previsibilidade.

3️⃣  TYPOGRAPHY É JOST* (SEMPRE)
   Google Fonts: 'Jost' (todos os pesos 100-900)
   Mesma fonte do site público — consistência!

4️⃣  ACESSIBILIDADE IMPORTA
   Contraste mínimo 4.5:1 (AA)
   Este design atinge 15.8:1 (AAA) — use!

5️⃣  TESTE EM DEVICE REAL
   Não apenas no desktop — teste em celular real
   Use Chrome DevTools para mobile simulation
   Redimensione a janela do navegador

6️⃣  REUTILIZE OS COMPONENTES
   Não reinvente a roda — use componentes do design system
   Isso garante consistência visual
