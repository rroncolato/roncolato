⚠️  MUDE CREDENCIAIS:

  .env:
    ADMIN_USER=seu-usuario-seguro-nao-admin
    ADMIN_PASS=senha-muito-forte-com-simbolos-12+chars

🔑 CONFIGURE reCAPTCHA v3 PRÓPRIO:

  1. Vá para: https://www.google.com/recaptcha/admin/create
  2. Crie novo site (reCAPTCHA v3)
  3. Domínio: seu-dominio.com
  4. Copie Site Key e Secret Key
  5. Atualize em /admin/login.html e .env

📤 FAÇA DEPLOY NO VERCEL:

  vercel env add JWT_SECRET
  vercel env add ADMIN_USER
  vercel env add ADMIN_PASS
  vercel env add RECAPTCHA_SECRET
  vercel --prod --yes
