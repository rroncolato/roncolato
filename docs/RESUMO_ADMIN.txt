## 🚀 Como Começar (5 minutos)

### 1. Configure as variáveis

Crie arquivo `.env` na raiz:
```
JWT_SECRET=sua-chave-super-secreta-minimo-32-chars
ADMIN_USER=admin
ADMIN_PASS=senha123
RECAPTCHA_SECRET=6LeIxAcTAAAAAGG-vFI1TnRWxMZNFuojJ4WifJWe
NODE_ENV=development
```

### 2. Acesse a área admin

**Desenvolvimento:**
```
http://localhost:8080/admin/login.html
```

**Ou centralized:**
```
http://localhost:8080/admin/index.html
```

### 3. Faça login

- Usuário: `admin`
- Senha: `senha123`

### 4. Publique conteúdo

- **Aba Portfólio:** Adicione projetos com imagens
- **Aba Blog:** Publique artigos com formatação

## 🔐 Segurança Implementada

| Camada | Proteção |
|--------|----------|
| **Frontend** | Validação de inputs + reCAPTCHA v3 |
| **Transporte** | HTTPS (Vercel automático) |
| **Autenticação** | JWT com HMAC-SHA256 |
| **Autorização** | Verificação de token em cada requisição |
| **Input Sanitization** | Escaping XSS + detecção de código malicioso |
| **Rate Limiting** | Vercel limita automaticamente |
| **CORS** | Apenas domínios autorizados |
| **Secrets** | Variáveis de ambiente (+gitignore) |

## 📤 Deploy no Vercel

```bash
# 1. Adicionar variáveis no Vercel
vercel env add JWT_SECRET
vercel env add ADMIN_USER
vercel env add ADMIN_PASS
vercel env add RECAPTCHA_SECRET

# 2. Deploy
vercel --prod --yes

# 3. Acessar
# https://seu-dominio.com/admin/login.html
```

## 📞 Problemas Comuns

### Erro "Token não fornecido"
→ Faça login novamente

### reCAPTCHA falha
→ Verifique Site Key e Secret Key em https://www.google.com/recaptcha/admin

### Projeto não salva
→ Verifique console (F12) para mensagens de erro

### Não conecta à API
→ Verificar `.env` e rodar `npm run dev`
